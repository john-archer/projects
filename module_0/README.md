# module_0
Проект в SkillFactory, модуль 0

Игра!

Суть: программно (псевдо)случайно выбираем число от 1 до 100. Задача — программно "угадать" это число, постаравшись совершить как можно меньше попыток.

Реализовано несколько функций поиска, от случайного блуждания до сужающейся воронки :) Для сравнения функций поиска между собой используется оценка, основанная на количестве попыток (в среднем) в серии испытаний (10 тысяч исходов) подобрать верное решение.
